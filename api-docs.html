🔌 API

API Documentation

Programmatic access to BSCAN's website audit engine.

🌐 Base URL
https://api-bscan.balancewises.io
🔑 Authentication

All authenticated endpoints require a Bearer token in the Authorization header.

Authorization: Bearer YOUR_API_KEY

API keys are available on Pro and Agency plans. Manage yours in the Account dashboard.

📡 Endpoints
POST /api/scan

Run a full website audit scan

🔒 Requires auth
POST /api/scan/check

Check scan allowance for an email

POST /api/scan/chat

Chat with AI about scan results

GET /api/scans/leaderboard

Get the public leaderboard

GET /api/scans/history

Get your scan history

🔒 Requires auth
POST /api/scan/compare

Compare two websites side-by-side

🔒 Requires auth
POST /api/seo/keywords

Keyword research for a topic

🔒 Requires auth
POST /api/seo/backlinks

Backlink analysis for a domain

🔒 Requires auth
POST /api/auth/login

Authenticate and get a token

POST /api/auth/register

Create a new account

GET /api/auth/me

Get current user info

🔒 Requires auth
POST /api/billing/checkout/:plan

Create a Stripe checkout session

GET /api/team

Get team members

🔒 Requires auth
POST /api/team/invite

Invite a team member

🔒 Requires auth

💻 Quick Start
curl -X POST https://api-bscan.balancewises.io/api/scan \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "url": "https://example.com",
    "email": "you@company.com"
  }'
